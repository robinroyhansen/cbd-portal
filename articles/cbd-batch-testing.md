# CBD Batch Testing: Why Every Batch Matters

By Robin Roy Krigslund-Hansen | 12+ years in CBD industry
Last updated: January 21, 2026

---

## Quick Summary

Batch testing means testing each production lot of CBD products separately, rather than relying on old test results. This practice ensures every product you buy has been verified for potency and safety, as natural products can vary between batches.

---

## Key Takeaways

- Each production batch should be tested
- Natural products vary between batches
- Batch numbers link products to test results
- Outdated tests don't reflect current product
- Quality brands test every batch

---

## Why Batches Vary

### Natural Variation

Hemp products vary because:
- Different harvest times
- Varying growing conditions
- Different plant batches
- Extraction variations
- Manufacturing differences

### What Can Change

**Between Batches:**
- CBD concentration
- Minor cannabinoid levels
- Terpene profiles
- Potential contaminants
- Overall composition

---

## What Is Batch Testing?

### Definition

Batch testing means:
- Each production lot tested individually
- Unique batch number assigned
- Separate COA for each batch
- Results specific to that production run

### Batch vs Single Test

| Approach | What It Means | Reliability |
|----------|---------------|-------------|
| Batch testing | Every lot tested | High |
| Periodic testing | Some lots tested | Medium |
| Single test | One-time only | Low |

## Identifying Batch Information

### On Product Labels

Look for:
- Batch number or lot number
- Manufacturing date
- Expiration date
- QR code linking to COA

### Common Formats

- Batch: CBD-2026-0124
- Lot: 20260124-001
- Production: PRD240124
- Various alphanumeric codes

### Matching to COA

**Verification Steps:**
1. Find batch number on product
2. Locate COA for that batch
3. Verify numbers match exactly
4. Check test date is appropriate
5. Review results

**Potency Testing:**

---

## Red Flags

### Warning Signs

**Generic COAs:**
- No batch number
- "Representative sample"
- Very old dates
- Same COA for all products

**Missing Information:**
- No batch on product
- Can't provide matching COA
- Batch format inconsistent
- No way to verify

**Poor Practices:**
- Testing every X batches
- Annual testing only
- In-house testing only
- No COA available

---

## Quality Brand Practices

### What to Expect

**Best Practices:**
- Every batch tested
- COA easily accessible
- Batch number clear on product
- QR code links to specific COA
- Recent test dates

**Transparency:**
- Explains testing practices
- Provides batch-specific results
- Responsive to questions

## How to Verify Batch Testing

### Steps to Take

1. **Check Product**
   - Locate batch/lot number
   - Note the format

2. **Find COA**
   - Scan QR code if present
   - Check company website
   - Request directly

3. **Match Information**
   - Batch numbers identical
   - Test date makes sense
   - Results complete

4. **Evaluate Results**
   - Potency within 10%
   - Contaminants pass
   - Lab accredited

## Questions to Ask Companies

### Before Purchasing

- Do you test every batch?
- How can I find my batch's COA?
- What do you test for?
- Which lab do you use?
- How recent is testing?

### If Something Seems Wrong

- Why doesn't my batch number match?
- When was this batch tested?
- Can you provide batch-specific results?
- What is your testing schedule?
